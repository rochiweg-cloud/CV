# Teaching Experience

I served as a Professor at [Educación.Earth](https://educacion.earth/), teaching the following professional courses:

## Deep Learning for Environmental Monitoring
Course focused on applying neural networks and deep learning techniques to environmental challenges:
- Environmental data processing and preparation
- CNN architectures for satellite imagery
- Time series forecasting for climate data
- Object detection for wildlife monitoring
- Transfer learning for environmental applications
- Case studies in climate change and conservation

## Computer Vision Workshop: Advanced Object Segmentation
Hands-on workshop covering modern segmentation techniques:
- Instance vs semantic segmentation
- Deep learning architectures for segmentation (U-Net, Mask R-CNN)
- Training segmentation models
- Working with environmental and satellite imagery
- Practical applications in land use classification
- Model deployment and scaling

## Geospatial Analysis and Machine Learning
Comprehensive course combining GIS and AI techniques:
- Advanced geospatial data processing
- Machine learning for geographic data
- Remote sensing analysis
- Spatial statistics and clustering
- Predictive modeling with geographic features
- Integration of satellite imagery and ground data
